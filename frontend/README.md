# 🎨 User Management Frontend

## 📌 About This Project
This is the frontend of the User Management System built using React.js. It provides a modern, responsive admin dashboard interface to manage users efficiently.

The application allows administrators to perform operations like adding, editing, deleting, and viewing users. It also includes advanced features like search, pagination, toast notifications, and modal-based editing for a better user experience.

---

## ✨ Features
- 🔍 Search users by name and email
- 📄 Pagination (10 users per page)
- ✏️ Edit user details using modal
- ❌ Delete users with confirmation
- 🔔 Toast notifications for actions
- 📱 Fully responsive dashboard UI
- 🎨 Clean and modern design

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- React Router DOM
- React Toastify
- Lucide React (Icons)
